  • 냉장고 재료 관리: 재료 등록/수정/삭제, 검색·수동 입력
  • 맞춤 레시피 추천: 재료+난이도 기반 추천, LLM 조리 가이드, 부족 재료 대체 제안
  • 식단 캘린더 & 타이머: 선택한 레시피 기록, 조리 타이머, 기록 기반 통계
  • 영양 루틴: 영양제 등록/체크, “오늘의 루틴 완료” 피드백
  • 대시보드 & 뱃지: 기간별 요리/카테고리 시각화, 습관 기반 뱃지

Cookus Architecture

GitHub Actions가 Frontend/Backend/Dashboard 이미지를 빌드하여 ECR에 저장하고, 각 EC2 인스턴스가 이미지를 Pull 해 Docker 컨테이너로 서비스합니다.

AWS Cloud

ECR + GitHub Actions 기반 CI/CD, 태그별 자동 배포

Front-end EC2

React 앱과 Streamlit 대시보드를 컨테이너 단위로 분리, Elastic IP로 노출

Back-end EC2

FastAPI/Uvicorn 컨테이너가 MariaDB · GPT-4o-mini · S3와 통신

데이터 계층

MariaDB + S3 Feature Store, Streamlit Ops 대시보드

지능 계층

GPT-4o-mini로 조리 가이드 요약/추론 및 부족 재료 대체 제안

아키텍처 정의 · React/Streamlit UI · GitHub Actions CI/CD 자동화를 직접 리드했습니다.
